# How this vault works

All reading notes can be found in `./Articles` or `./Papers` folders. Files properties are normalized between them to allow better graph render.

## Notes

### What should be a note ?

- Reading note
- Notions that need a definition (ASD, Habituation, ...)
- Topics that can be found in other papers (VR, Guidelines, )
- Questionnaires (ADOS, ...)

All notes should begin with an uppercase letter.
### Reference a note

To reference another note, please write

`[[Reference note]]`
>This is said in the previous note

For the sake of quality, all notes contents that are not in the `./Synthesis and protocols` folder should only be references. However, due to the amount of relevant elements this can be delayed or just have a link to an article.
